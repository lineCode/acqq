# acqq

> 本项目使用vue2.0高仿腾讯动漫的实战项目，先上传简单版本，后续改动会陆续更新

### 项目技术框架
* vue2.0
* vue-cli
* vue-resource
* vue-route
* vuex
* mint-ui
* swiper
* less
* webpack

### 安装

安装依赖
```
npm install
```

```
npm run dev
```
